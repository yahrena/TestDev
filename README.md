# TestDev

###Clonez le dépôt :
```sh
git clonne https://github.com/yahrena/TestDev.git
```

# backend

## Installation

```sh
npm install
```
### Configurez les variables d'environnement:
en créant un fichier .env basé sur le modèle fourni.

### Démarrez le serveur backend

```sh
npm start
```




# frontend
